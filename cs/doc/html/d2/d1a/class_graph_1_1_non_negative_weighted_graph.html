Detailed Description

a weighted graph implementation based on the Graph Class forcing non-negative weights.

See also
WeightedGraph A weighted Graph assigns a weight to an edge. This weight can be used by algorithms to find the most opimal paths, such as Dijkstra's or A* algorithms. Forcing non negative weights makes sure that we can always use Dijkstras
