\doxysubsection{Detailed Description}
a \doxylink{class_mazes_1_1_graph}{Graph} Implementation for non null types. 

A graph is a set of vertices with edges in between. Given this relationship, we can create a structure that can later be visualized and implemented for other purposes. Some Uses would be \doxylink{namespace_mazes}{Mazes}\+: Both Generation and Solving, as well as relational pathing between 2 objects. Implements IEnumerable in order to be able to loop through the vertices. \begin{Desc}
